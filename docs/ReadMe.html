This README file gives a general overview of the ManyMouse project.

Instructions for building ManyMouse are in the BuildInstructions.html file in the docs folder.

Required APIs

ManyMouse depends on the RawInput API for Windows as well as the Allegro 5 graphics library. The RawInput API allows an application to receive raw input from hid compliant devices such as keyboards and mice. Allegro 5 allows you to create and draw to a window as well as load images and draw graphics and more.
